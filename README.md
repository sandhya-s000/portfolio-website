# 🌐 HTML5 Semantic Structure & Accessibility Portfolio

## 📌 Project Overview

This project is a responsive and accessible personal portfolio website developed using **HTML5** and **CSS3**. It demonstrates the use of semantic HTML elements, accessibility best practices, SEO-friendly structure, and responsive web design.

## 🚀 Features

- Semantic HTML5 structure
- Responsive design
- Accessible navigation
- Contact form with proper labels
- SEO-friendly meta tags
- Mobile-friendly layout
- Clean and modern user interface
- Multi-page website
